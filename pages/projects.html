LangNotes (2022)

View on Github

Personal portfolio project

This project only purpose is to display my skills and knowledge in web development

This application has been designed as a tool to support you learn a new language. As the name itself suggests, LangNotes is a notebook for language learning, allowing the user to insert lessons and vocabularies.
Lessons can contain notes on some specific argument; for instance, in a French class I could have a Lesson called "Form of greetings" that could contain the various examples (e.g. Salut - Hello, Bon jour - Good morning, etc).
The full application has been developed with PERN stack languages (PostgreSQL, Express, React and NodeJS).
